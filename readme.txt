Modify email texts used in your WooCommerce emails without having to edit email templates.

== Description ==
This plugin allows you to edit email texts that are used in WooCommerce customer emails without having to edit the default templates that WooCommerce uses for emails.

Unlike other plugins, it will not overwrite your existing templates.

== Installation ==
1. Install and activate the plugin through the plugins menu.
2. Visit WooCommerce > Settings > Emails and select a customer email. Youâ€™ll find a new field where you can edit the content of an email.
